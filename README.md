# CET-6 翻译知识点笔记

本项目整理了大学英语六级（CET-6）翻译模块的核心知识点、常见句型、高频词汇与翻译技巧，并以单页 Web 应用的形式呈现，方便查阅与复习。

## 在线预览

直接在浏览器中打开 `web/index.html` 即可查看笔记页面。

## 内容来源

- **整理者**：糯糯不爱吃糖
- **授课老师**：四六级邹老师

笔记内容根据四六级邹老师的授课内容整理制作，版权归原作者与整理者所有。

## 项目结构

```
.
├── web/
│   └── index.html          # 单页笔记网页（主要阅读入口）
├── 翻译.md                  # Markdown 格式源文件
├── 翻译.pdf                 # PDF 格式源文件
├── 翻译词汇/                # 翻译常考词汇
└── *.png                    # 讲义截图
```
